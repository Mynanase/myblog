# 立绘

主要是绘制带有不同表情的半身照. 基本流程

  1. SD 生成合适的个人半身照
  2. 利用面部 ControlNet 生成简单的动作
  3. 利用局部重绘绘制表情
  4. 使用 SD 无损放大
  5. 导入 pixian ai 去除背景
  6. 导入 PS 精修, 保持整体风格一致 ( 找个美工 )

如此绘制的立绘整体比较呆板, 没有什么动作. 可以尝试将我们的照片和已存在 galgame 的照片融合生图.
